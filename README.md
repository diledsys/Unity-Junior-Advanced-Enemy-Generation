# Unity-Junior-Advanced-Enemy-Generation
## Описание
Учебный проект на Unity.
Продолжаем работать над проектом с генерацией врагов. 

Теперь каждая точка спауна создает свой тип врагов и имеет цель, то есть объект, к которому идут враги. В момент создания точка спауна передает врагу свою цель, чтобы тот направился к ней. Таких целей должно быть несколько, у каждого спавнера своя цель.

Сами цели движутся по заранее определенному маршруту, от одной точки к другой.

## Используемые технологии
- Unity 6.3LTS
- C#

## Структура
- Assets/_Project/Scripts — логика управления
- Assets/_Project/Scenes — основная сцена
- Asset/_Project/Prefab — Prefab

## EnemySpawner

- Создание врагов определённого типа и назначение им конкретной цели. 
- Спавнит enemyPrefab с заданным интервалом.
- Ограничивает количество активных врагов (maxAlive).
- Передаёт врагу цель через SetTarget.
- Уничтожает врагов по таймеру жизни.
- Поддерживает корректный счётчик живых врагов.
 
##  EnemyChaseTarget

- Преследование назначенной цели.
- Получает цель через SetTarget.
- Поворачивается в сторону цели.
- Двигается к цели через CharacterController.
- Останавливается на stopDistance.
- Обновляет параметры анимации (Speed, MotionSpeed).
- Обрабатывает гравитацию вручную.
 
##  TargetWaypointMover
 
- Движение цели по заданному маршруту.
- Ходит по массиву waypoints.
- Поддерживает режимы loop / ping-pong.
- Поворачивается по направлению движения.
- Использует CharacterController.
- Применяет гравитацию.
- Обновляет анимацию.
 
##  TargetRegistry

- Регистрация всех целей для систем, которым нужен их список (например, камеры).
- Находит все объекты с TargetWaypointMover.
- Передаёт список в CameraGroupFramer или другую систему.
- Позволяет добавлять/удалять цели динамически.
 
- Архитектурная схема
 
- Spawner → создаёт Enemy → назначает Target
- Target → движется по маршруту
- Enemy → преследует назначенный Target
- Registry → агрегирует список целей для внешних систем (камера и т.п.)

## Как запустить
1. Открыть проект в Unity Hub
2. Загрузить сцену Main
3. Нажать Play
